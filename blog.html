Important FAQ

What is the Document Object Model (DOM)?

In order to make a website interactive, JavaScript connected with HTML creates a tree structured model which represents the whole HTML document in an object that includes all the tags as nodes for which it becomes easier to locate any element through JavaScript and make it functionable. This is called Document Object Model.

How do you select an element from the DOM?

Using the DOM created by JavaScript, an element can be selected by typing document.getElementById() or document.getElementsByClassName() or document.getElementByTagName() depending on the type of element and given class or id. There are also querySelector() and querySelectorAll() to select first and all elements respectively in a parent container.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in JavaScript means if the child elements in a parent container need to be changed or manipulated, instead of adding Event Listeners separately to all the child elements, one event listener is added to the parent container by which the childs can be controlled.

How do you manipulate an element's attributes and styles using the DOM?

By selecting the element by its id and giving it a variable, using the .setAttribute(), .removeAttribute() and .getAttribute(), the attributes of that exact element can be changed or the value can be seen. Likewise, adding .style with the variable and setting a property and value, the styles of that element can also be given or changed.
